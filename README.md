# @yeunoia/bloom

Wrap anything with bloom. → [See Bloom](https://yeunoia-bloom.vercel.app/)

## Install

```
npm install @yeunoia/bloom
```

## Usage

```tsx
import { Bloom } from "@yeunoia/bloom";

<Bloom>highlighted words</Bloom>

```

## Props

| Prop | Type | Default |
|------|------|---------|
| `type` | `"box" \| "line"` | `"box"` |
| `tip` | `"round" \| "square"` | `"round"` |
| `backgroundColor` | `string` | `"#A4E7D5B3"` |
| `color` | `string` | `"inherit"` |
| `paddingX` | `number` | `4` |
| `paddingY` | `number` | `2` |

## Requirements

React ≥ 17
