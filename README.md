# @yeunoia/smear

Wrap anything with smear. → [See Smear](https://yeunoia-smear.vercel.app/)

## Install

```
npm install @yeunoia/smear
```

## Usage

```tsx
import { Smear } from "@yeunoia/smear";

<Smear>highlighted words</Smear>

```

## Props

| Prop | Type | Default |
|------|------|---------|
| `type` | `"box" \| "line"` | `"box"` |
| `tip` | `"round" \| "square"` | `"round"` |
| `backgroundColor` | `string` | `"#A4E7D5B3"` |
| `color` | `string` | `"inherit"` |
| `paddingX` | `number` | `4` |
| `paddingY` | `number` | `2` |

## Requirements

React ≥ 17
