# @yeunoia/smear
[Wrap anything with smear](https://yeunoia-smear.vercel.app/)

## Props

| Prop              | Type                | Default |
| ----------------- | ------------------- |----|
| `backgroundColor` | `string`            | `#A4E7D5B3` |
| `color`           | `string`            | `inherit` |
| `tip`             | `"flat" \| "round"` | `"round"` |
| `paddingX`        | `number`            | `4` |
| `paddingY`        | `number`            | `2` |

`tip="flat"` — highlighter-like edges  
`tip="round"` — brush-like edges

## Requirements

React ≥ 17
